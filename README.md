# Clean Up the Environment
### An Educational Arcade Game Built with Pygame

Clean Up the Environment is a simple yet meaningful 2D arcade-style game developed using Python and Pygame. The game encourages environmental responsibility by tasking players with collecting falling trash before it reaches the ground.

Designed as an awareness-driven project, it blends basic game mechanics with a sustainability message, making learning interactive and engaging.

---

## Overview

Environmental pollution is a growing concern, and awareness can begin through small, engaging experiences.  
In this game, the player controls a character that moves horizontally to collect falling trash. Missing too much trash results in a game over, reinforcing the importance of timely action and responsibility.

---

## Gameplay Mechanics

- Move the player left and right using keyboard controls
- Catch falling trash to increase score
- Each missed trash increases the miss counter
- The game ends after 5 missed trash items
- Final message emphasizes environmental conservation

---

## Features

- Interactive arcade-style gameplay
- Collision detection using bounding boxes
- Score and miss tracking
- Start screen with instructions
- Game-over and final awareness message
- Image-based sprites and background

## Controls

- Left Arrow Key: Move player left  
- Right Arrow Key: Move player right  
- Any Key: Start the game
